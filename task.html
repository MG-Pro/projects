Тема:

Вычисление частот гласных и согласных в тексте.

Проект будет реализован с использованием следующих технологий:

  • Javascript
  • HTML
  • CSS
  • AJAX
  • PHP

Обеспечивается поддержка русского и английского алфавитов

Клиентская часть

Веб страница будет содержать текстовое поле для ввода текста и кнопку "Отправить". Пользователь вставляет или набирает текст в поле и нажимает кнопку. Если поле пустое, кнопка не активна, требуется текст длинной не менее 3 символов. После будет произведен запрос к серверу, получен ответ в формате JSON и выведен результат в отформатированном виде. Чтобы не ограничивать размер текста, запрос будет передаваться методом POST.

Серверная часть

Сервер будет обрабатывать POST запрос в котором содержится текст и возвращать ответ в формате JSON содержащий количество гласных и согласных букв. Будут учитываться буквы из английского и русского алфавитов одновременно.
